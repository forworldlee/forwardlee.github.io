##分区类型
RANGE 连续的列值区间分区，有主键和唯一键必须使用，如果没有则可以指定任何一列；
LIST 类似RANGE分区，区别在于指定一系列的列值作为分区条件；
HASH 由函数表达式返回值决定所在分区，函数返回值必须为非负整数；
KEY 由mysql提供的HASH函数进行服务，==使用较少==。
